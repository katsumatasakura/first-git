在職中身につけたこと

自走する力

周りに同じことをやっている人はおらず、チームプレイよりは個人プレイ。自分で進捗を管理して業務を行っていました。

目的意識を持つ力

自分の仕事は何のためにあるか、やろうとしていることは目的に沿っているか。常に自問しながら日々の仕事に取り組んでいます。

説明する力

予算折衝の仕事では、根拠のある説明を求められてきました。納得感を引き出すための思考法が身についています。

学習で身につけたこと

Photoshop / Illustrator

バナー作成、ロゴ作成、Webサイトのデザインカンプ作成ができます。ツールを使う段階から、コーディングでの実現可能性を考慮しつつデザインしています。

HTML / CSS

SEOを意識したHTMLのマークアップができます。CSSは、コメントを活用しチームで開発しやすく修正にも強い記述を心がけています。

jQuery

ハンバーガーメニュー、カルーセルなどの動きをつけることができます。実現したい動きを自分で記述できるように、その都度調べて吸収しています。

学習中のこと

Adobe XD

デザインカンプの作成ができます。UIデザイナーを目指したいので、プロトタイプも作成できるよう勉強中です。

Git

実務ではバージョン管理をしながら開発するのが一般的かと思ったので、ツールを使いながら慣れようとしています。

JavaScript

基本的な文法を学んだ上で、Reactなどのライブラリを通してモダンなJavaScriptの学習にも取り組んでいるところです。
